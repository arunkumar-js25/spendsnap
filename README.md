# 🚀 SpendSnap

**SpendSnap** is a smart expense tracking app that simplifies daily expense management using QR scanning and intelligent insights.

---

## ✨ Features

### 📷 QR-Based Expense Tracking

* Scan UPI QR codes
* Automatically extracts:

  * Amount 💰
  * Description 📝
* Eliminates manual entry

---

### ⚡ Auto Expense Creation

* Pre-fills expense details after scan
* One-tap save experience
* Fast and user-friendly workflow

---
 
### 🧠 Smart Category Detection

* Automatically categorizes expenses based on description
* Examples:

  * Food 🍔
  * Travel 🚗
  * Shopping 🛍️

---

### 📊 Insights Dashboard

* Total expense tracking
* Category-wise breakdown
* Weekly comparison:

  * “You spent more than last week”
  * “You saved compared to last week”

---

### 💾 Local Storage (Offline First)

* Uses SQLite (Drift)
* Works without internet
* Fast and reliable data persistence

---

## 📱 Screens

* Home Screen (Expenses + Insights)
* Add Expense Screen
* QR Scanner Screen

---

## 🛠️ Tech Stack

* **Flutter** – Cross-platform UI
* **Drift (SQLite)** – Local database
* **Mobile Scanner** – QR scanning
* **Riverpod (planned)** – State management

---

## 🔄 App Flow

1. Open app
2. Scan QR or add expense manually
3. Auto-fill details
4. Save expense
5. View insights instantly

---

## 🎯 Key Highlight

> Scan → Auto-fill → Auto-category → Save

SpendSnap reduces friction in expense tracking and makes the process effortless.
